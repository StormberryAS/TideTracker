# TideTracker

Privacy-first coastal tides & currents predictor. TideTracker computes expected high and low tides, lunar influence, and visualises the water level for any coastal city on Earth.

**Live:** [tide.stormberry.as](https://tide.stormberry.as)

## Features
- **City search**: rapid, offline autocomplete for 2,000+ major cities (re-uses SunApp/MoonApp dataset).
- **On-device geolocation**: retrieve your current coordinates with one click.
- **Tide Graph**: visual water-level animation using CSS gradients.
- **Moon Influence**: links tide data to the lunar phase (Spring/Neap tides).
- **Privacy first**: fully client-side.

## Architecture
- **Vanilla HTML/CSS/JS**, no frameworks, no build step.
- Stormberry dark-mode glassmorphism design system.
